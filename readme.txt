== Description ==

AVENAZ D-Gallery is easies plugin to show all the images in a directory in jQuery Masonary using easy to use short code.

You can use this plugin into pages and posts easily 

Just need to specify the directory path into short and its all done

You can upload bulk images via ftp or other files managers and this plugin 


== Installation ==

1. Install plugin via admin or uplaod and install it
2. Activate Plugin
3. Once installed use the short code [folder width="image_width_here" folder="folder_path_here"]
folder path should start from be in wp-contents like if your folder path an example is

[folder width="125" folder="wp-contents/photoz"]
